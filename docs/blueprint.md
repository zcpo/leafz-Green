# **App Name**: GreenLeaf Reviews

## Core Features:

- Location Navigation: Navigate from Home to State to City to Dispensary pages.
- Review Submission: Allow users to submit reviews, ratings, and photos for dispensaries.
- Firestore Integration: Store location and review data in Firestore using a hierarchical structure.
- User Authentication: Implement user authentication using Firebase Email and Google login.
- Dispensary Mapping: Show dispensaries on Google Maps with location markers.
- Admin Panel: Provide an admin interface for easy dispensary creation and editing.

## Style Guidelines:

- Primary color: Black (#000000), a strong, contrasting color.
- Background color: White (#FFFFFF), providing a clean and neutral base.
- Accent color: Sky Blue (#87CEEB), providing a fresh and airy touch, which highlights and balances the black.
- Body and headline font: 'Inter', sans-serif. A versatile sans-serif for a clean and modern user experience.
- Use simple, consistent icons throughout the app, in the Apple/Yelp style.
- Implement a mobile-first, responsive design to mimic the Apple/Yelp interface.
- Use subtle, non-intrusive animations to enhance user interaction.
